
# 📘 Assignment: Hangman Game Challenge

## 🎯 Objective

Build a Hangman game in Python that uses string manipulation, loops, conditionals, and user input to create an interactive guessing experience.

## 📝 Tasks

### 🛠️ Create the game loop

#### Description
Implement the main Hangman game loop to select a word, display the current guessed state, and accept user letter guesses.

#### Requirements
Completed program should:

- Randomly select a word from a predefined list
- Display blanks for the hidden word and reveal letters as they are guessed
- Accept single-letter guesses from the user
- Track and show the number of incorrect guesses remaining

### 🛠️ Add win/lose game flow

#### Description
Detect win and lose conditions, then show the correct result when the game ends.

#### Requirements
Completed program should:

- End the game when the word is fully guessed
- End the game when the player has used all allowed incorrect guesses
- Display a win message when the player guesses the word
- Display a lose message and reveal the correct word when attempts are exhausted
