# 📘 Assignment: Intro to Classification with scikit-learn (Iris)

## 🎯 Objective

Train and evaluate a simple supervised classification model using scikit-learn on the Iris dataset. Students will learn train/test splitting, model training, evaluation metrics, and interpreting a confusion matrix.

## 📝 Tasks

### 🛠️ Setup and exploratory data analysis

#### Description
Load the Iris dataset, inspect features, and visualize class distributions.

#### Requirements
Completed work should:

- Load the Iris dataset using scikit-learn or from CSV
- Display the first few rows and basic summary statistics
- Plot a pairwise scatter plot or feature histograms showing class separation

### 🛠️ Train and evaluate a classifier

#### Description
Train a classification model and evaluate it using common metrics.

#### Requirements
Completed program should:

- Split data into train and test sets (e.g., 80/20)
- Train a `LogisticRegression` or `KNeighborsClassifier` model
- Report accuracy, precision, recall (or F1) on the test set
- Display a confusion matrix for the test predictions

### 🛠️ Interpret results and short report

#### Description
Analyze model performance and suggest one improvement.

#### Requirements
Completed submission should:

- Provide a short written summary (2–4 sentences) interpreting results
- Suggest one realistic step to improve the model (feature engineering, different model, cross-validation)

## Starter files

- `starter-code.py`: minimal script that loads data and trains a basic model (provided)
- `requirements.txt`: required packages for the assignment

## Submission

Students should submit the completed script or a Jupyter notebook and include the brief report as a README section or notebook cell.
